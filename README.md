很多人想做 AI 项目，但真正卡住的不是模型 API，而是：

- 不知道做什么项目
- 不知道第一版 MVP 应该做到哪里
- 不知道技术栈怎么选
- 不知道怎样把项目写进简历、作品集或 GitHub
- 不知道一个想法有没有继续做下去的价值

这个仓库不是简单收集“点子”，而是把每个 AI 项目拆成可以直接开干的方案。

## 适合谁

- 想做 AI 开源项目的开发者
- 想准备简历项目 / 毕设 / 作品集的学生
- 想用 AI 做副业或小产品验证的独立开发者
- 想学习 AI Agent、RAG、MCP、LLM App 的工程实践者
- 想找 GitHub 项目方向但不想再做烂大街 Demo 的人

## 精选项目

### 1. AI GitHub Profile Maker

输入 GitHub 用户名，AI 自动生成 Profile README、个人定位、Pinned 仓库建议和涨粉计划。

- 难度：中等
- 关键词：GitHub API、LLM、Profile README、个人品牌
- 适合人群：开源作者、求职者、技术博主
- 详情：[ideas/ai-github-profile-maker.md](ideas/ai-github-profile-maker.md)

### 2. Repo Reader Agent

输入一个 GitHub 仓库地址，AI 帮你读懂项目结构、启动方式、技术栈和核心模块。

- 难度：中等
- 关键词：Repository Analysis、Code Reading、Agent、RAG
- 适合人群：开源学习者、代码审查者、技术写作者
- 详情：[ideas/repo-reader-agent.md](ideas/repo-reader-agent.md)

### 3. AI Code Reviewer

读取 Git diff 或 Pull Request，AI 自动生成代码审查报告，指出风险、坏味道和可改进点。

- 难度：中等偏上
- 关键词：Code Review、GitHub Actions、Diff Analysis、Security
- 适合人群：开发团队、开源维护者、工程效率工具作者
- 详情：[ideas/ai-code-reviewer.md](ideas/ai-code-reviewer.md)

### 4. AI Resume Optimizer

上传简历和目标 JD，AI 分析匹配度、优化项目描述、生成面试准备清单。

- 难度：简单到中等
- 关键词：Resume、JD Matching、Career、LLM App
- 适合人群：求职者、应届生、培训机构、技术博主
- 详情：[ideas/ai-resume-optimizer.md](ideas/ai-resume-optimizer.md)

### 5. Agent Tools Lab

用真实代码讲清楚 AI Agent 如何安全、可控、可扩展地调用工具，并接入 MCP。

- 难度：中等
- 关键词：AI Agent、Tool Calling、MCP、Permission
- 适合人群：AI 应用开发者、Agent 学习者、工程化实践者
- 详情：[ideas/agent-tools-lab.md](ideas/agent-tools-lab.md)

## 每个 Idea 包含什么

每个项目方案都会尽量包含：

- 一句话介绍
- 目标用户
- 真实痛点
- MVP 功能范围
- 推荐技术栈
- 页面 / 模块结构
- 数据结构建议
- 实现步骤
- 可扩展方向
- 简历包装方式
- 开源传播标题

## 如何使用这个仓库

1. 先从精选项目里挑一个你最有兴趣的方向。
2. 根据 MVP 范围做第一版，不要一开始做大平台。
3. 把开发过程写成文章或视频。
4. 项目上线后把反馈沉淀回仓库。
5. 如果某个 idea 反馈好，再升级成独立开源项目。

- 解决什么痛点

- 为什么值得做
