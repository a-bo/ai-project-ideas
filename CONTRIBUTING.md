# Contributing

欢迎贡献新的 AI 项目创意。

为了保证这个仓库不是“标题合集”，请不要只提交项目名。一个合格的 idea 至少需要回答：

- 谁会用？
- 解决什么痛点？
- 第一版 MVP 做什么？
- 推荐技术栈是什么？
- 为什么值得做？
- 可以如何传播？

## 提交方式

1. 复制模板：[templates/idea-template.md](templates/idea-template.md)
2. 在 `ideas/` 目录新增 Markdown 文件
3. 在 `ideas/index.md` 中加入索引
4. 发起 Pull Request

## 好 idea 的标准

- 场景具体
- 用户明确
- 第一版可做小
- 有传播标题
- 能写进简历或作品集
- 有后续扩展空间

## 不建议提交

- 只有一句话的点子
- 纯套壳 ChatGPT
- 没有目标用户
- 没有实现路径
- 已经严重同质化且没有差异化说明
