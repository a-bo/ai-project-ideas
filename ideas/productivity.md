# Productivity

效率类 AI 项目适合做小产品，容易上线、容易验证需求。

## 1. AI Weekly Report Generator

根据工作记录、Git commit、任务列表自动生成日报、周报、月报。

MVP:

- 输入零散工作记录
- 自动分类工作内容
- 生成周报
- 生成下周计划
- 支持 Markdown 导出

传播标题：

```text
再也不怕写周报：AI 自动把工作记录整理成周报
```

## 2. AI Meeting Summary

把会议录音或文字纪要整理成摘要、行动项、责任人和截止时间。

MVP:

- 输入会议文本
- 提取议题
- 提取结论
- 提取行动项
- 输出飞书 / Markdown 格式

## 3. AI Email Reply Assistant

根据邮件上下文生成礼貌、专业、符合语气的回复。

MVP:

- 输入原邮件
- 选择回复意图
- 选择语气
- 生成中英文回复
- 生成多个版本

## 4. AI Personal Knowledge Base

把个人笔记、PDF、网页收藏变成可问答的知识库。

MVP:

- 上传 Markdown / PDF / TXT
- 文档切分
- 向量检索
- 问答
- 引用来源

## 5. AI Task Planner

把一个模糊目标拆成可执行任务、时间计划和检查清单。

MVP:

- 输入目标
- 生成里程碑
- 拆分任务
- 估算优先级
- 导出 Todo
